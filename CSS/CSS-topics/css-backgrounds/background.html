CSS - font-variant - small-caps

This value is used to display the text in small caps. It converts lowercase letters to uppercase letters while keeping their size smaller than the regular uppercase letters.

CSS - font-variant - normal

It's worth noting that browser support for font-variant and its values may vary, and small caps might not be available for all fonts. Additionally, the font-variant property is often used in combination with other font properties to achieve the desired text styling.
