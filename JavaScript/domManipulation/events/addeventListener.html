Add event listener

The addEventListener() is an inbuilt function in JavaScript which takes the event to listen for, and a second argument to be called whenever the described event gets fired. Any number of event handlers can be added to a single element without overwriting existing event handlers.

element.addEventListener(event, function, options);
Here, event is a string representing the type of event to listen for (e.g., "click", "input", "change"), and function is the event handler function to be executed when the event is detected.
The optional options parameter is an object that can be used to specify additional options for the event handling.
